# Spring Boot Kotlin - VueJS Test Project

FullStack Kotlin (JVM) / Spring / H2 / VueJS testing app.
Feel free to fork :)

### How-to

Run the app with

```
$  cd kotlin-back
$ ./gradlew bootRun
```

### State

**Back**

- HelloWorld on `/hello` path
- REST Api on `beers`
